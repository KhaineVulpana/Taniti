# Taniti Tourism Website - Interactive Prototype

This is an interactive HTML/CSS/JavaScript prototype for the Taniti tourism website redesign project.

## Files Included
- `index.html` - Homepage
- `activities.html` - Things to Do page
- `stay.html` - Where to Stay page
- `plan.html` - Plan Your Trip page
- `styles.css` - Complete stylesheet
- `script.js` - Interactive functionality

## Features Implemented
Based on guerrilla usability testing feedback:

1. **Homepage Improvements**
   - Added "Welcome to Taniti" introductory section explaining what Taniti is
   - Clear navigation to main sections

2. **Things to Do Page**
   - Activities organized into clear categories (Beaches, Rainforest Adventures, Volcano & Culture)
   - Family-friendly indicators with visual tags
   - Pricing information ($, $$, $$$) for all activities
   - Age requirements clearly displayed
   - Working filters for category and price

3. **Where to Stay Page**
   - Pricing ranges for all accommodation types
   - Budget to luxury options ($, $$, $$$, $$$$)
   - Clear categorization by type
   - Working filters for type, price, and location

4. **Plan Your Trip Page**
   - Transportation costs prominently displayed with budget indicators
   - Cost comparison between buses, taxis, and rental cars
   - Dining cost information
   - Safety and medical services information added
   - Sample itineraries

5. **Mobile Responsive**
   - Fully functional on mobile devices
   - Hamburger menu for navigation
   - Touch-friendly buttons and links
